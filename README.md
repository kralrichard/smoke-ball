# Smoke Ball

Smoke Ball is a casual mobile game developed with Unity.

## 🎮 Game Description
Throw the smoke ball, aim carefully, and complete levels by reaching the target.
The game focuses on simple mechanics and fast gameplay.

## 🛠 Built With
- Unity
- C#

## 📱 Platform
- Android
